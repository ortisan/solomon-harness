# Educational Psychologist Definition of Done

This skill defines the criteria that all educational psychologist deliverables must meet to be considered complete.

## Design Criteria
- Every learning design proposal must explicitly cite a methodology from the closed allow-list.
- Learning outcomes must be designed before instructional materials are created.
- Instructional text must avoid cognitive overload by presenting information in clean, bite-sized segments.
- Visual and verbal information must be combined to support retention.
- All assessments must align with the target learning objectives.

## Verification
- propuestas are reviewed against the established criteria.
- No generalist platitudes or unscientific fads are included in the recommendations.

## Common pitfalls

- A deliverable accepted because it "matches the learner's visual style" — learning styles failed empirical review (Pashler et al., 2008) and never justifies a design.
- Instructional text, slides, or video drafted before the learning outcomes and rubric exist — backward design requires objectives and evidence first, so activity-first work is rejected.
- Engagement, satisfaction, or completion metrics presented as evidence of learning — only performance against the stated objectives counts as evidence.
- An assessment offering recognition (multiple choice on syntax) when the objective demands recall or production (write the code) — constructive alignment is broken.
- A cited technique with an effect size below the d >= 0.40 threshold, or with no peer-reviewed source at all — it fails the fad-rejection criterion.
- Sign-off given without screening the proposal against the closed allow-list, letting folk pedagogy pass as common sense.

## Definition of done

- [ ] Every design decision names its allow-list methodology inline, with the citation placed next to the decision it justifies.
- [ ] Learning objectives use measurable verbs (write, analyze, debug) and were documented, with their rubric, before any instructional material was drafted.
- [ ] Each unit stays within the four-chunk working-memory limit and pairs every abstract concept with a diagram whose text is integrated rather than redundant.
- [ ] Every assessment task requires the exact behavior stated in its objective; no recall objective is tested with a recognition item.
- [ ] The review record shows the proposal was screened for fads and platitudes, noting any rejected technique and its failing effect size.
- [ ] Retention mechanics are explicit where the program spans days: recall prompts at the six-minute cadence and reviews on the 1/7/30-day spacing schedule.
