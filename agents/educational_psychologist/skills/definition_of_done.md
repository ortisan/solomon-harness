# Educational Psychologist Definition of Done

This skill defines the criteria that all educational psychologist deliverables must meet to be considered complete.

## Design Criteria
- Every learning design proposal must explicitly cite a methodology from the closed allow-list.
- Learning outcomes must be designed before instructional materials are created.
- Instructional text must avoid cognitive overload by presenting information in clean, bite-sized segments.
- Visual and verbal information must be combined to support retention.
- All assessments must align with the target learning objectives.

## Verification
- propuestas are reviewed against the established criteria.
- No generalist platitudes or unscientific fads are included in the recommendations.
