# Statistical Modeling

Purpose: Verify statistical hypotheses and calculate confidence intervals.

## Core Rules

1. Define Null Hypothesis Clearly
   - Before executing any hypothesis test, explicitly declare the Null Hypothesis ($H_0$), Alternative Hypothesis ($H_a$), significance level ($\alpha$, usually 0.05), and chosen test statistical model (e.g. t-test, Chi-square).

2. Report p-values and Confidence Intervals
   - Always accompany test outcomes with calculated p-values and confidence intervals to define prediction margins of error.

3. Verify Model Assumptions
   - Verify statistical assumptions (e.g., normality, homoscedasticity, independence) before accepting linear regression or statistical test outputs, executing transformations if assumptions are violated.
