# Reporting and Visualization

Purpose: Extract insights from data and represent them in structured reports and visual stories.

## Core Rules

1. Define the Core Metric First
   - Start every report by declaring the key performance indicator (KPI) being analyzed, explaining its business significance.

2. Visual Clarity
   - Do not clutter dashboards or charts. Use appropriate chart types (e.g. line charts for trends over time, bar charts for comparisons, scatter plots for correlations).
   - Ensure labels, units, and axes are explicit and easily readable.

3. Structured Narrative (Data Storytelling)
   - Follow a logical sequence: context, data observation, root cause analysis, and actionable business recommendations.
